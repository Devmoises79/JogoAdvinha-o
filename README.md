## 🎯 Jogo de Adivinhação (C# Console)

Projeto simples em C# onde o usuário tenta adivinhar um número gerado pelo sistema. O programa fornece feedback se o chute é maior ou menor até o acerto.

# 🎮 Funcionalidades
- Geração de número aleatório
- Entrada de palpites via console
- Feedback se o número é maior ou menor
- Contagem de tentativas até acertar
- Loop até vitória do jogador

# 🧠 Conceitos aplicados
- Estruturas de repetição (while / do-while)
- Condicionais (if / else)
- Random em C#
- Entrada e saída de dados no console
- Lógica de comparação

# 🚀 Como executar
dotnet run

# 🛠️ Tecnologias
- C#
- .NET Console Application

# 👨‍💻 Objetivo
Projeto desenvolvido para praticar lógica de programação, manipulação de números aleatórios e estruturas de repetição.
